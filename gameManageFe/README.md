本课题采用SSM和ajax技术框架进行开发，后台采用MySql数据库系统。
系统主要用于向游戏玩家群体提供一个可方便查看游戏战队、队员数据和游戏对战记录，并对对战记录和游戏战队，队员进行评价交流的系统。
系统分为游客（只能浏览信息）、会员（可以评论交流）和管理员（后台管理）三种用户类型。

## 主要前台功能包括:
### 系统信息管理：会员注册，登陆，修改个人信息和登陆密码,登出。
### 战队信息查看：查看游戏战队信息，也可查看一个战队旗下包括的所有队员。会员可以对战队信息进行留言和点赞。
### 队员信息查看：查看队员的个人信息以及所属战队。会员可以对队员信息进行留言和点赞。
### 游戏对战记录查看：包括对战信息浏览（只显示胜负以及战队名称）和对战详细信息查看（在对战记录浏览中，点击某一局对战记录会跳转到对战详细信息（包括两支战队的名称以及上场队员及使用英雄）。会员可以对对战记录进行留言和点赞。
### 对战录像观看：在对战信息浏览表单，可以点击观看比赛的录像，并且可以下载录像。

主要后台管理功能包括：
战队信息管理：战队信息的新增修改和删除。
队员信息管理：队员信息的新增修改和删除。
对战记录管理：对战记录的新增修改和删除。
比赛录像上传系统:可以选择对战记录对应的录像并且上传。
会员信息管理：管理会员的数据，查看游客用户的登录记录，可以封禁游客用户。
留言信息管理：对战队、队员和对战记录的留言和点赞进行管理。
